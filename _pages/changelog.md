## **Version 1.4.7**
June 13 , 2020

Today we have these changes for you

- The unread badge is now smarter and will disappear. (In some cases you need to send one more message to make it work)

- Photos will now upload faster (Did you know that we delete every photo after one day? We take privacy seriously)

- The map will only zoom automatically, when a friends checks in

- We fixed an issue where the delete account button didn't delete everything


Please leave a review if you are happy with the changes :)

<br>

## **Version 1.4.6**
June 9 , 2020

We got some news for you. This update includes these improvements:

- Your bitmojis will now update

- There are now bitmojis in the chat

- Now we show the snapchat name in a friend request

- We've added a delete friend button when you tap on a friend on the map

- The map will not randomly move around anymore. It will do it only when a friends checks in

- You can now opt out of the suggestest friends list and the user search in the advanced settings

- It's now possible to delete a sent friend request


Please leave a review if you are happy with the changes :)
